لوحة إدارة الأسئلة

هذه الصفحة تتيح إضافة وتعديل وحذف الأسئلة المخزنة في Supabase. تأكد من تهيئة window.SUPABASE_URL و window.SUPABASE_KEY قبل تحميل السكربت أدناه.

قائمة الأسئلة

إضافة سؤال جديد / تعديل

ملاحظات: Admin عبر المتصفح يتطلب مفتاح Supabase العام/المنشئ؛ لأمان أفضل ضع هذه الواجهة خلف مصادقة أو نفّذ CRUD بواسطة API آمن.
